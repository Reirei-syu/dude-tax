# PRD - dude-tax

- 产品显示名：工资薪金个税计算器

## 背景
面向财务或薪酬使用场景，提供单位维度的工资薪金个税管理工具。系统需要长期保存历年数据，同时允许快速测算与正式结果分离。

## 核心用户
- 财务人员
- 薪酬专员
- 需要快速核验个税结果的内部使用者

## 核心价值
- 降低手工核算与核对成本
- 将“单位 + 年份”作为统一工作房间，减少误操作
- 统一全局税率标准，保证口径一致

## 当前优先级
### P0
- 工程初始化
- 单位管理
- 年份上下文
- 首页上下文展示

### P1
- 员工信息
- 月度录入
- 计算中心
- 结果中心

### P2
- 快速计算
- 历史查询
- 导入导出
- 系统维护

## 成功标准
- 软件可以在本机启动
- 可创建单位、切换单位
- 可切换年份
- 前端能通过 API 读取和修改上下文
- 数据能在 SQLite 中持久化
- 可执行年度重算，并在结果中心查看或手动切换年度结果方案
- 可在结果中心查看当前员工年度结果的历史版本时间线
- 可在结果中心对两个历史版本进行关键字段差异对比
- 可在结果中心更清晰地管理导出模板，查看模板摘要并获得导出反馈
- 可查看年度结果的导出预览字段，为后续导出功能做准备
- 可按导出模板或所选字段导出当前年度结果 CSV / XLSX，完成最小结果交付能力
- 可通过 CSV / XLSX 双格式交付当前年度结果，满足初步对外输出需求
- 可录入月度已预扣税额，并在结果中心查看年度应纳税额、已预扣与应补/应退
- 个税计算核心已具备预扣预缴情形共享类型、月度轨迹纯函数和年度预扣轨迹摘要
- 可在月度录入页按季度查看全年 12 个月状态，并快速切换到任意月份编辑
- 可在月度录入页将当前表单批量应用到多个月份，形成未保存的本地草稿
- 可在月度录入页将所选草稿批量保存，完成多月份录入闭环
- 可在月度录入页提示本地草稿与已保存记录之间的字段差异
- 可在历史查询页按单位、年份、员工和结算方向检索历史年度结果
- 可在历史查询页切换查看当前有效结果、已失效快照和全部结果
- 可在历史查询页导出当前筛选结果 CSV / XLSX
- 可在历史查询页对已失效快照按当前税率进行结果差异对比
- 可在历史查询页查看所选员工年度结果的重算版本历史
- 可在系统维护页查看、编辑和保存当前税率与全局富文本说明，并在税率变更时逻辑失效旧结果
- 可在系统维护页看到税率草稿校验问题、字段高亮提示，并在问题修正前禁止保存
- 可在系统维护页查看历史税率版本并激活旧版本
- 可在系统维护页将历史税率版本绑定到当前单位 / 年度，并让结果按该作用域税率判定有效性
- 可在系统维护页查看税率变更审计日志
- 可在系统维护页解除当前单位 / 年度的专属税率绑定，并恢复继承全局活动版本
- 可在系统维护页预览目标税率版本与当前作用域版本的关键差异及结果影响
- 可在快速计算页录入临时月份数据，通过本地 API 完成不落库测算并查看双方案结果
- 可在批量导入页下载 CSV 模板，上传 CSV / Excel 文件，查看细粒度预览结果、处理冲突并执行导入
- 可在首页看到批量导入冲突摘要，并按当前状态获得更细粒度的模块入口建议
- 可在结果中心看到按结算方向生成的更完整结算说明
- 可在结果中心和历史查询中看到规则来源摘要，明确是否引用跨单位 / 跨年度规则上下文
- 可通过 `npm run package:win` 生成 Windows 测试包目录，桌面壳会启动编译后的本地 API，并将数据库落到用户数据目录
