# PRD - 工资薪金个税计算器

## 背景
面向财务或薪酬使用场景，提供单位维度的工资薪金个税管理工具。系统需要长期保存历年数据，同时允许快速测算与正式结果分离。

## 核心用户
- 财务人员
- 薪酬专员
- 需要快速核验个税结果的内部使用者

## 核心价值
- 降低手工核算与核对成本
- 将“单位 + 年份”作为统一工作房间，减少误操作
- 统一全局税率标准，保证口径一致

## 当前优先级
### P0
- 工程初始化
- 单位管理
- 年份上下文
- 首页上下文展示

### P1
- 员工信息
- 月度录入
- 计算中心
- 结果中心

### P2
- 快速计算
- 历史查询
- 导入导出
- 系统维护

## 成功标准
- 软件可以在本机启动
- 可创建单位、切换单位
- 可切换年份
- 前端能通过 API 读取和修改上下文
- 数据能在 SQLite 中持久化
