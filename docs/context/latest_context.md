## 当前阶段
- 项目处于 `v0.1.0-alpha`
- 项目已初步完成，主要业务模块、前后台数据勾连和模块联动均已跑通
- 当前重点转向复杂税务口径、导出策略细化、数据围栏和边界收口

## 已完成模块
- Electron + React + Fastify + SQLite 本地桌面骨架可运行
- “单位 + 年份”全局上下文闭环已接通
- 员工信息、月度录入、计算中心、结果中心、历史查询、系统维护、快速计算、批量导入均已具备最小可运行闭环
- 系统维护已支持税率编辑、富文本说明维护、版本激活、单位 / 年度作用域绑定和逻辑失效判定
- 首页已联动待重算、未完成月份、导入冲突和工作建议
- 历史查询已支持结果筛选、状态切换、跨年度概览、CSV / XLSX 导出、失效快照差异对比和重算版本历史
- 结果中心已支持历史版本查看、版本差异对比、导出模板摘要和导出反馈
- 批量导入已支持 CSV、XLSX、XLSM 文件读取，以及细粒度预览展示
