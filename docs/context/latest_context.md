## 已完成模块
- Electron + React + Fastify + SQLite 本地桌面骨架可运行
- “单位 + 年份”全局上下文闭环已接通
- 员工信息、月度录入、计算中心、结果中心、历史查询、系统维护、快速计算、批量导入均已具备最小可运行闭环

- 本轮已完成本地 API 来源限制、Electron 窗口安全加固和导入 CSV 复杂字段解析增强

## 当前任务
- 本轮已完成 `全项目审查 -> 安全加固 + 导入链路稳健性增强`
- 已收紧本地 API 暴露面，并为导入链路补上复杂 CSV 字段解析与测试

## 关键设计决策
- 补发补扣字段当前只进入共享类型和 helper，不直接并入现有算法主链
- 补发补扣首版仍按“支付当月补发 / 调整”建模，不先做往期申报更正工作流
- 继续保持 `UI -> Service/API -> Core` 分层，不在前端直接拼后端业务规则
- 本地 API 仅允许 Electron / 本地开发来源访问，外部站点一律返回 `403`
- Electron 窗口禁止打开新窗口和导航到外部页面，外链统一交给系统浏览器
