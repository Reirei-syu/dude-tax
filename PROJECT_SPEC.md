# dude-tax - 项目规格

- 产品显示名：工资薪金个税计算器

## 1. 产品目标

- 提供单位级、年度级的工资薪金个税录入、确认、查询与导出能力
- 采用桌面端本地存储方式，支持历年数据长期保存
- 提供快速计算模块，用于不落库的临时试算

## 2. 顶层业务模型

- 全局工作上下文：`单位 + 年份`
- 全局政策上下文：`税率标准 + 政策参考说明`
- 正式业务主线：
  - 员工信息维护
  - 月度数据录入
  - 结果确认
  - 历史查询
- 临时业务主线：
  - 快速计算
  - 仅用于试算，不写入正式月度记录

## 3. 当前模块清单

- 首页
- 单位管理
- 员工信息
- 快速计算
- 月度数据录入
- 批量导入
- 结果确认
- 历史查询
- 政策参考
- 系统维护

## 4. 当前功能边界

- 月度数据录入
  - 一级卡片展示年度员工列表
  - 纳入当年任意时点在职员工
  - 支持月份多选统计预扣税额
  - 支持进入可最大化年度工作台编辑 12 个月数据
  - 工作台收入区使用“工资收入、年终奖、预扣税额、其他收入、其他收入备注”
  - 工作台不再显示“记录状态”
  - 工作台展示只读“减除费用”列，值来自当前政策 `basicDeductionAmount`
  - 支持将当前月应用到下月 / 后续月份
  - 不提供执行计算按钮
  - 支持导出“汇总 sheet + 每员工 sheet”Excel
- 快速计算
  - 采用单案例总览 + 年度工作台
  - 不填写姓名等基础信息
  - 与月度录入使用同一套工作台字段
  - 支持将当前月应用到下月 / 后续月份
  - 支持基于当前单位 / 年份政策即时试算
- 结果确认
  - 合并原计算中心与结果中心
  - 按月份顺序确认
  - 确认后锁定对应月份数据
  - 取消确认时级联解除当前月及后续月份
  - 允许空白月直接确认
  - 已确认但无录入内容的月份按零值月参与计算
  - 列表与明细只展示已确认数据
  - 支持导出“汇总 sheet + 每员工 sheet”Excel
- 历史查询
  - 只查询已确认数据
  - 条件保留单位、年份
  - 展示年度员工纳税列表与全年明细
  - 支持导出“汇总 sheet + 每员工 sheet”Excel

## 5. 当前数据与接口约束

- 数据库保留旧 `status`、补扣税调整、所属期间字段用于兼容，不做破坏性删列
- 新工作台、新 API 与新计算逻辑不再依赖这些旧字段
- `supplementary_salary_income` 兼容映射到 `otherIncome`
- `supplementary_remark` 兼容映射到 `otherIncomeRemark`
- `supplementary_withheld_tax_adjustment` 与 `supplementary_source_period_label` 在新主流程中视为废弃字段

## 6. 关键非功能要求

- 全离线运行
- Windows 优先
- 桌面 UI 适配常见 PC 分辨率
- 所有关键路径可测试、可回退
- 确认后的月份数据不可被普通录入流程修改
