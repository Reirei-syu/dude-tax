# 工资薪金个税计算器 - 项目规格

## 1. 产品目标
- 提供单位级、年度级的工资薪金个税录入、计算、查询、导出能力
- 采用桌面端本地存储方式，支持历年数据长期保存
- 提供快速计算模块，用于不落库的即时测算

## 2. 顶层业务模型
- 全局工作上下文：`单位 + 年份`
- 全局政策上下文：`税率标准 + 提示说明`
- 正式业务流：员工数据维护 -> 计算 -> 结果查看/导出
- 临时业务流：快速计算 -> 仅展示结果，不写入数据库

## 3. 模块清单
- 首页
- 单位管理
- 员工信息
- 月度数据录入
- 批量导入
- 计算中心
- 结果中心
- 历史查询
- 系统维护

## 4. 当前首个开发里程碑
- Electron + React + Fastify + SQLite 项目骨架可运行
- 首页可展示当前上下文和占位工作提醒
- 单位管理支持新增、选择、删除
- 年份切换可用，并可全局驱动页面状态
- 本地 API、数据库、桌面壳打通

## 5. 关键非功能要求
- 全离线运行
- Windows 优先
- 桌面 UI 自适应常见 PC 分辨率
- 浅色现代风格
- 所有关键路径可测试、可回退
