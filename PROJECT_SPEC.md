# dude-tax - 项目规格

- 产品显示名：工资薪金个税计算器

## 1. 产品目标
- 提供单位级、年度级的工资薪金个税录入、计算、查询、导出能力
- 采用桌面端本地存储方式，支持历年数据长期保存
- 提供快速计算模块，用于不落库的即时测算

## 2. 顶层业务模型
- 全局工作上下文：`单位 + 年份`
- 全局政策上下文：`税率标准 + 提示说明`
- 正式业务流：员工数据维护 -> 计算 -> 结果查看/导出
- 临时业务流：快速计算 -> 仅展示结果，不写入数据库

## 3. 模块清单
- 首页
- 单位管理
- 员工信息
- 月度数据录入
- 批量导入
- 计算中心
- 结果中心
- 历史查询
- 系统维护

## 4. 当前开发里程碑
- Electron + React + Fastify + SQLite 项目骨架可运行
- 首页可展示当前上下文和占位工作提醒
- 首页工作提醒已联动“待重算”和“未完成月份”统计
- 首页已根据当前状态给出工作建议入口
- 单位管理支持新增、选择、删除
- 年份切换可用，并可全局驱动页面状态
- 员工信息模块支持当前单位下的最小 CRUD
- 月度数据录入支持当前单位/年份/员工下的 12 个月基础维护
- 月度数据录入支持按季度分组的整年视图、快速切月与单月编辑联动
- 月度数据录入支持将当前表单批量应用到多个月份的本地草稿
- 月度数据录入支持将所选草稿批量保存到数据库
- 月度数据录入支持提示本地草稿与已保存记录之间的数据差异
- 计算中心支持按员工/按年度的准备状态查看与真实年度重算
- 结果中心支持查看已生成的年度结果、手动方案切换与双方案对比
- 结果中心支持查看当前员工年度结果的历史版本时间线
- 结果中心支持对两个历史版本进行关键字段差异对比
- 结果中心支持导出模板、字段分组选择，并可按所选字段导出当前年度 CSV / XLSX 文件
- 结果导出当前已具备 CSV / XLSX 双格式交付能力
- 月度录入支持维护“已预扣税额”，结果中心支持展示年度应纳税额、已预扣税额与应补/应退结果
- 历史查询支持按单位、年份、员工和结算方向查看历史年度结果
- 历史查询支持切换查看当前有效结果、已失效快照和全部结果
- 历史查询支持导出当前筛选结果 CSV / XLSX
- 历史查询支持对已失效快照按当前税率进行结果差异对比
- 历史查询支持查看所选员工年度结果的重算版本历史
- 系统维护支持查看、编辑和保存当前税率与全局说明，并在保存税率后逻辑失效旧结果
- 系统维护支持前端税率草稿校验、字段高亮提示和保存前禁用校验
- 系统维护支持查看税率版本列表并激活历史版本
- 系统维护支持将历史税率版本绑定到当前单位 / 年度作用域
- 快速计算支持按当前单位 / 年度税率进行不落库测算，并展示双方案结果
- 批量导入支持 CSV 模板下载、CSV / Excel 文件导入、细粒度预览、冲突处理和执行导入
- 首页工作提醒支持联动导入冲突摘要，工作建议支持按导入冲突、待导入数据、月度缺口和待重算状态分发入口
- 结果中心支持按结算方向生成更完整的结算说明
- 本地 API、数据库、桌面壳打通

## 5. 关键非功能要求
- 全离线运行
- Windows 优先
- 桌面 UI 自适应常见 PC 分辨率
- 浅色现代风格
- 所有关键路径可测试、可回退
